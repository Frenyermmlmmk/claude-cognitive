# Local Development - Task API

> **Role**: Local development environment
> **Host**: `localhost`
> **Hardware**: Developer machine
> **Critical Path**: No - local testing only

## Topology
| Direction | Connected To | Protocol | Purpose |
|-----------|--------------|----------|---------|
| ← Receives | Browser/curl | HTTP:8000 | API requests |
| → Sends | SQLite file | Local file | Data persistence |

## Quick Health
```bash
curl http://localhost:8000/health

```

## Key Processes
- `main.py`: FastAPI server on port 8000
